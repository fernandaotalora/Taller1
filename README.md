# Taller1

El código de la aplicación se encuentra en el archivo de nombre Taller1

Al ejecutar la aplicación toma como archivo de entrada los contenidos en la carpeta 
archivoinicial
los archivos de los resultados de vectores se guardan en la carpeta en 
archivosprocesado

Para realizar la busqueda se debe ejecutar la funcion: vectorSearch
la cual muestra una ventana que solicita el ingreso de las palabras a buscar y
un boton para iniciar el proceso de busqueda
Luego de finalizar la busqueda se muestra el listado de las noticias que coincidan con 
la busqueda
